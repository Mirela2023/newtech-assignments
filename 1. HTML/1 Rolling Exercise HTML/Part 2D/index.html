Codul binar

Ce este codul binar, Nov 20, 2020

Calculatorul poate sa retina doar 1 si 0.
Codul binar este un limbaj utilizat la calculatoare. Se bazează pe sistemul numeric binar, care folosește doar 0 și 1 ca simboluri pentru a reprezenta un text. Toate calculatoarele și platformele digitale utilizează codul binar pentru a procesa instrucțiuni și a lucra eficient.
